How It Works

1. Create or Join a Tree

From this home page, either create a new tree for your family or join an existing tree with an access code. Once you’re inside a tree, everything you see is specific to that family only.

2. View Your Family Tree

On the "Family Tree" page, you’ll see everyone in the current tree organized by generation, with spouses and siblings grouped together.

Go to Family Tree

3. Add a Family Member

On the Family Tree page, click the green "+" button in the bottom-right corner to add someone new to this tree.

  • Required: First Name and Last Name
  • Optional: Middle Initial, Birth Date, Parents, Spouse
  • Parents: Enter names as "First Last" (e.g., "John Smith")
  • Spouse: Also as "First Last" (e.g., "Jane Doe")

4. View & Edit Profiles

Click on any person’s card in the tree to open their profile. You’ll see:

  • Birth date
  • Parents and children
  • Spouse information
  • A historical fun fact based on their birthday

From the profile page, you can edit or remove that person from the current family tree.

5. Search for Family Members

Use the "Search" page to quickly find family members by name within the currently selected family tree.
